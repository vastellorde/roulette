# Тестовое задание

Приложение каталог фильмов

## Возможности
* Возможность сделать ставку, выбирая число, кратность или цвет
* Возможность провести розыгрыш
* Частично адаптивная
* lazy loading

## Запуск сборка
1) Сборка `ng build`
2) Запуск `ng serve`

## Используемые инструменты
* angular 10
* @angular/components
* rxjs
* scss
